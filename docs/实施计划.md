# 薪日 PayDay - 实施计划

实施计划按 **Phase** 关联以下文档，开发与验收时按 Phase 查阅对应文档。

**验收**：各 Phase 的验收统一放在该 Phase 后期（见 [迭代规划_Sprint与任务.md](迭代规划_Sprint与任务.md) 各 Phase 最后一轮 Sprint），当前以功能实施为主。

**当前阶段**：Phase 2 社区版，按 Sprint 2.1 实施（帖子与列表、发帖页、帖子详情）。

## 文档查阅顺序建议

1. **做某个 Phase**：先看 [迭代规划.md](迭代规划.md) 该 Phase 的验收标准，再看 [迭代规划_Sprint与任务.md](迭代规划_Sprint与任务.md) 的 Sprint 任务。
2. **做某个模块的接口/表/缓存**：查 [技术方案_按模块细化.md](技术方案_按模块细化.md) 对应模块。
3. **查全局架构或约定**：查 [技术方案_v1.0.md](技术方案_v1.0.md)。
