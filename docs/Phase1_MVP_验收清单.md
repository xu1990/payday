# Phase1 MVP 验收清单

依据 `迭代规划.md` § 1.4 MVP验收标准整理，用于上线前逐项勾选。

---

## 一、功能验收

### 小程序端
- [ ] 用户可以完成微信登录
- [ ] 用户可以配置发薪日（支持多工作）
- [ ] 用户可以记录工资（支持图片上传）
- [ ] 首页显示发薪倒计时和状态
- [ ] 可以查看工资统计（月度/年度）
- [ ] 可以生成分享海报

### 管理后台
- [ ] 管理员可以登录
- [ ] 可以查看用户列表与用户详情
- [ ] 可以查看工资记录列表（支持按用户 ID 筛选）
- [ ] 可以删除单条工资记录
- [ ] 可以对单条工资记录进行人工审核（通过/拒绝）
- [ ] 可以查看基础统计（用户总数、今日新增、工资记录总数、今日记录数）

---

## 二、性能验收
- [ ] 接口响应时间 < 500ms (P95)
- [ ] 页面加载时间 < 2s
- [ ] 支持 500 并发用户

---

## 三、安全验收
- [ ] 工资金额加密存储
- [ ] 图片上传安全检测
- [ ] 接口基础限流

---

## 四、上线准备
- [ ] 小程序提交审核
- [ ] 服务器部署完成
- [ ] 域名备案完成
- [ ] 监控告警配置
